##### `fmt`


##### 格式化动词

通用格式化动词

| 动词 | 描述                         |
| ---- | ---------------------------- |
| %v   | 值的默认格式，可打印任何类型 |
| %T   | 打印值的类型                 |

布尔与整数动词

| 动词 | 描述             |
| ---- | ---------------- |
| %d   | 十进制整数       |
| %b   | 二进制           |
| %o   | 八进制           |
| %x   | 十六进制         |
| %X   | 十六进制（大写） |
| %t   | 布尔值           |

浮点数与复数动词

| 动词 | 描述              |
| ---- | ----------------- |
| %g   | 根据精度选择%e/%f |
| %e   | 科学计数法（e）   |
| %E   | 科学计数法（E）   |
| %f   | 小数形式          |

字符串与字节动词

| 动词 | 描述                                 |
| ---- | ------------------------------------ |
| %s   | 字符串                               |
| %q   | 带引号的字符串（对特殊字符进行转义） |
| %x   |                                      |

|      |      |
